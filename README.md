# Telegram Bot для заметок

Простой Telegram-бот для управления заметками с функцией напоминаний.

## Функционал

- ✅ Добавление заметок
- ✅ Просмотр всех заметок
- ✅ Удаление заметки по номеру
- ✅ Удаление всех заметок
- ✅ Установка напоминаний с указанием даты и времени

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ULFRIC21/tg.git
cd tg
```

2. Установите зависимости:
```bash
pip install aiogram
```

3. Настройте токен бота в `main.py`:
```python
bot = Bot(token='YOUR_BOT_TOKEN')
```

4. Запустите бота:
```bash
python main.py
```

## Использование

- `/start` - начать работу с ботом
- `/help` - показать справку

### Создание заметки с напоминанием

1. Нажмите "Добавить заметку"
2. Введите текст заметки
3. Введите дату и время в формате: `ДД.ММ.ГГГГ ЧЧ:ММ` (например: `25.12.2024 15:30`)
   - Или напишите "пропустить" чтобы сохранить без напоминания
